# Sahte Avcısı - Web Demo

Sahte Avcısı, oyuncunun şüpheli ürünleri inceleyerek gerçek, sahte veya şüpheli kararı verdiği tarayıcı tabanlı bir ekspertiz simülasyonu prototipidir.

## Oyun Mantığı

Oyuncu bir ekspertiz ofisi işletir. Müşteriler şüpheli ürünlerle gelir. Oyuncu seri numarası, fatura ve fiziksel inceleme gibi araçları kullanarak kanıt toplar ve karar verir.

## Özellikler

- 10 farklı vaka
- Para ve itibar sistemi
- Kanıt toplamadan karar verememe
- Gerçek / Sahte / Şüpheli karar sistemi
- Sonuç ekranı
- Performans unvanı

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript

## Durum

Bu proje erken aşama bir web demo prototipidir. Geliştirme süreci devam etmektedir.

## Not

Bu proje kişisel öğrenme ve prototipleme amacıyla geliştirilmiştir. Geliştirme sürecinde yapay zeka destekli kodlama araçlarından yardım alınmıştır.
